Hi! I'm Ianis Arquissandas

I'm a Game Developer and Programmer in Portugal. I'm currently at my last year in college, taking a Videogame Bachelor degree, at Universidade Lusófona. I've been in love with games since I can remember, as such I decided to pursue a career related to game developing.

I learned as much as I could using the Unity Engine, I also learned some programming languages, such as some basic knowledge in C and JavaScript, and a very extensive knowledge of C# which is my main programming language.
